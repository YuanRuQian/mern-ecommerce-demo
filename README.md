# FEESH: MERN + AWS Tinned Fish E-Commerce Demo

## Frontend
- React SPA
- Redux
- React Router

## Backend
- Node.js
- Express
- MongoDB

## Deployment
- AWS EC2 (Ubuntu) hosting for backend
- AWS S3 for image storage & frontend hosting

## Try it out!
- [Live Demo](http://feesh-react-frontend.s3-website.us-east-2.amazonaws.com/)
